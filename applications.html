Open source applications built on NANDA infrastructure

Our commitment to open source drives everything we build. These applications demonstrate the capabilities of decentralized AI systems built on top of the NANDA index.

NANDA Applications

  • Agentic Chat

    A new kind of conversation platform where you stop chatting with AI and start chatting through AI. Your personal AI agent works invisibly in the background, making every message more thoughtful, clear, and kind.

  • NANDA Registry

    Decentralized registry interface for agent discovery and capability verification. Browse and interact with agents registered in the NANDA network.

  • NANDA Host

    Self-hosted registry node deployment tools. Contribute to network resilience by running your own NANDA registry instance.
